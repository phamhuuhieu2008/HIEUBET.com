# HIEUBET.com - Game Tài Xỉu Trực Tuyến

Chào mừng bạn đến với dự án HIEUBET. Đây là ứng dụng trò chơi Tài Xỉu (Sic Bo) được xây dựng trên nền tảng Web.

## ✨ Tính năng chính
- Hệ thống Đăng ký/Đăng nhập bảo mật.
- Giao diện 3D Dice với hiệu ứng chuyên nghiệp.
- Hệ thống Admin điều phối kết quả và duyệt giao dịch.
- Quản lý số dư, lịch sử đặt cược và rút tiền.

## 🚀 Công nghệ
- **Backend:** Node.js, Express
- **Frontend:** HTML, CSS, JavaScript (Tailwind CSS)
- **Database:** JSON File System (Lưu trữ cục bộ tại thư mục /data)
---
**Lưu ý quan trọng:**
Dự án này sử dụng hệ thống tệp JSON để lưu trữ dữ liệu. Mọi thay đổi về tài khoản, nạp/rút tiền sẽ được ghi trực tiếp vào thư mục `data/` trên máy tính của bạn. Đảm bảo bạn chạy server bằng lệnh `node server.js` để dữ liệu được đồng bộ chính xác.

## 🛠️ Cài đặt
1. `npm install`
2. `node server.js`
3. Truy cập `http://localhost:3000`
