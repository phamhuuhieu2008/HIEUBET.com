# HIEUBET.com - Game Tài Xỉu Trực Tuyến

Chào mừng bạn đến với dự án HIEUBET. Đây là ứng dụng trò chơi Tài Xỉu (Sic Bo) được xây dựng trên nền tảng Web.

## ✨ Tính năng chính
- Hệ thống Đăng ký/Đăng nhập bảo mật.
- Giao diện 3D Dice với hiệu ứng chuyên nghiệp.
- Hệ thống Admin điều phối kết quả và duyệt giao dịch.
- Quản lý số dư, lịch sử đặt cược và rút tiền.

## 🚀 Công nghệ
- **Backend:** Node.js, Express
- **Frontend:** HTML, CSS, JavaScript (Tailwind CSS)
- **Database:** JSON File System (Có thể nâng cấp lên MongoDB)
---
**Lưu ý quan trọng:**
Dự án này sử dụng hệ thống tệp JSON để lưu trữ dữ liệu người dùng. Nếu bạn triển khai trên Render hoặc các dịch vụ lưu trữ tạm thời, dữ liệu có thể bị xóa sau mỗi lần khởi động lại server. Khuyến khích sử dụng MongoDB cho môi trường thực tế.

## 🛠️ Cài đặt
1. `npm install`
2. `node server.js`
3. Truy cập `http://localhost:10000`
